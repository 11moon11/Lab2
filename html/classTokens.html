Detailed Description

Class to divide up a String into individual tokens based on a character delimiter.

The Tokens class accepts a String and a character (the delimiter) through the constructor. Several Strings (tokens) are created based on the appearance of the delimiter in the original String. The delimiter itself is not part of the tokens. The resulting tokens can be obtained by index according to their location in the original String. The original String is unaffected. Note that the destructor does not delete the individual tokens.
